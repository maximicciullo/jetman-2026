# Imágenes del Evento

## Cómo agregar las imágenes

### 1. Imagen del evento (Hero)
1. Coloca tu imagen del evento en esta carpeta con el nombre: `jetman-hero.jpg`
2. Formatos soportados: JPG, PNG, WebP
3. Tamaño recomendado: 1920x1080px o mayor (formato horizontal)
4. La imagen se mostrará como fondo del hero section de la landing page

### 2. Mapa de ubicación
1. Coloca la captura del mapa de Google Maps con el nombre: `jetpark-map.jpg`
2. Formatos soportados: JPG, PNG, WebP
3. Tamaño recomendado: 800x400px o mayor (formato horizontal)
4. La imagen se mostrará como preview del mapa (clickeable para abrir Google Maps completo)

## Nombres de archivos

- **jetman-hero.jpg** - Imagen principal del evento
- **jetpark-map.jpg** - Captura del mapa de ubicación

Si tus imágenes tienen otros nombres, puedes:
- Renombrarlas a los nombres indicados arriba
- O actualizar las rutas en `app/page.tsx`
